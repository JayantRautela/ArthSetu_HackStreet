# Problem Statement
Many Indian citizens struggle with:

- Choosing between Old and New tax regimes

- Understanding available deductions

- Calculating taxable income correctly

- Discovering relevant government schemes

- Interpreting financial terminology

- Receiving personalized financial advice

Existing platforms often:

- Provide raw numbers without explanation

- Lack personalization

- Do not integrate AI assistance

- Are difficult for non-experts to understand

<br />



# Our Solution
ArthSetu provides a structured, AI-enhanced financial decision system.

- Tax Regime Comparison Engine

  - Computes deductions (80C, 80D, Home Loan, etc.)

  - Calculates taxable income

  - Applies official Indian tax slabs

  - Computes effective tax rate

  - Determines potential savings

  - Recommends the optimal tax regime

  - Generates AI-based explanation

- Government Scheme Eligibility Engine

  - Rule-based eligibility system

  - Match score calculation

  - Eligibility explanation

  - Transparent rule evaluation

- AI Financial Assistant

  - Streaming responses

  - Markdown-formatted output

  - Budget planning tables

  - Tax advisory explanations

  - Real-time financial guidance
